Webscraping Color Contrast Checker Tool

I had made a webscraping tool that pulled the color of elements and compared them to the background color (contrast) and exported the results into a spreadsheet. The reason I made this tool is because when it comes to ADA compliance this is normally checked manually, and at the time I had a list of websites that needed to pass these issues, so I made a tool that did it automatically. I wrote this application using Python at first and then I switched over to C#/.NET.
